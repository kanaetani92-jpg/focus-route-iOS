# Build 9: Route Editor UX

## 変更内容

- ルート編集画面にカテゴリ入力を追加
- カテゴリはルート整理用として保存し、ホームと到着したルートに表示
- 「このルートの使い方」に判断基準の説明を追加
- テンプレート選択時にカテゴリを自動入力
- タスク編集行のレイアウトを調整
  - タスク名を主情報として広く表示
  - 時間と分割方法を短い入力欄に分離
  - 分割方法は「ステップ」付きの副情報として表示
- ルート複製時にカテゴリも引き継ぎ

## 意図

カテゴリはタイマーや進捗に影響させず、保存したルートを探しやすくするための分類として扱います。
「このルートの使い方」は、完了まで進める単発ルートか、日報・朝の準備のように繰り返し使う型かを選ぶための設定です。

タスク編集では、スマホ幅でタスク名と分割方法が同じ重さで並ぶと読みにくいため、
タスク名を優先し、分割方法は小さく確認できる形に整理しました。
