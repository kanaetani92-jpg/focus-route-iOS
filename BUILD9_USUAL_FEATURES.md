# Build 9: 通常文言の追加機能

## 追加した機能

- ホームに「すぐ始める」を追加
  - 「15分だけ集中」「5分」「25分」から、ルート作成なしで集中を開始できます。
- ホームに「テンプレート」を追加
  - 資料作成、勉強、片づけ、事務作業からルートを作れます。
- ルート作成に「カテゴリ」を追加
  - 仕事、勉強、暮らしを選べます。
- 集中画面に「今日はここまで」を追加
  - 途中までの実際の集中時間を記録し、ホームへ戻れます。
- ホームと一覧に「かかった時間」を追加
  - 記録された実際の集中時間を確認できます。

## 通常文言への調整

- 「今日のルート」を「今日やること」に変更しました。
- 「進んだ記録」を「完了記録」に変更しました。
- 「到着したルート」を「完了したルート」に変更しました。
- 集中画面の開始ボタンを「開始」「再開」に整理しました。
