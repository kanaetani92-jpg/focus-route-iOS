# Focus Route iOS Build 5

## タスク完了後の操作

タスク完了時の操作を、重要度順に次のように配置しました。

1. タスク完了の状態表示
2. 次のタスクへ進む
3. 前のタスク
4. ＋ もう1回分追加
5. このタスクをやり直す

## もう1回分追加

- 完了したタスクと同じ1回分の時間とステップ数を追加
- 例: 10分・2ステップのタスクなら、10分・2ステップを追加
- 2回目以降も最初の1回分だけを追加し、追加済み全体を倍増させない
- 追加前に時間とステップ数を確認
- 「追加して始める」で追加分のタイマーを自動開始
- Build 6以降、円は追加後も元の1回分と同じ分割数で表示
- 既に完了したステップと実測時間は維持
- 追加分は現在の実行だけに適用し、「もう一度使う」では元のタスク構成へ戻す
- 追加後10秒間は「元に戻す」が可能

## やり直す

- ボタン名を「このタスクをやり直す」に統一
- 実行前に確認画面を表示
- 完了済みステップを取り消し、タスクの先頭へ戻す
