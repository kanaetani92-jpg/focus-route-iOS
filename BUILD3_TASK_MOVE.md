# Focus Route iOS Build 3

## タスク移動時の選択

「前のタスク」「次のタスク」を押すと、現在のタスクについて次から選べます。

- 保留にして前／次へ
  - 現在位置と実測時間を保存
  - 未完了ステップを残す
  - 戻ったときは続きから再開
- 完了にして前／次へ
  - 残りのステップを完了として記録
  - 現在までの実測時間は保持
  - 未実施の残りステップの実測時間は0分として記録
- キャンセル
  - タスクを移動しない
  - 移動前にタイマーが動いていた場合は再開

## 誤操作対策

- 移動ボタンを押しただけでは移動しない
- 選択ダイアログに現在のタスク、完了ステップ数、移動先を表示
- 完了にする残りステップ数を具体的に表示
- 処理中の連続操作を禁止
- 移動後10秒間「元に戻す」を表示
- 元に戻すと、ルート位置、ステップ、経過時間、完了記録、保留状態を一括復元
